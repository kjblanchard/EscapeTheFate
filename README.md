# Escape The Fate


- Cross Platform 2d deckbuilder roguelike RPG demo
- Custom engine written in C.
- Lua scripting.
- Opengl 3.3 and OpenglES 3.0

## Details

- BGMs created by me in Garageband

## Goals

- Cross Platform CMAKE
- Package with CMAKE for installing on desktops
- C/Lua practice
- Learn OpenGL

## Installation

- Run any of the make commands for your platform, or run cmake directly,
    currently this bundles the demo game and engine together

    ```bash
    make rebuild
    ```
